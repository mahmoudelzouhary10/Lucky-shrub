# Lucky Shrub Website

This project was created as part of the Meta Front-End Developer course
(HTML & CSS in Depth).

The goal of the project is to build a simple and clean website using only
HTML and CSS, with a focus on layout, structure, and basic styling.

---

## Project Overview

The website represents a fictional company called Lucky Shrub.
It includes a hero section, a services section built using CSS Grid,
a navigation bar, and a footer.

---

## Technologies Used

- HTML5
- CSS3

---

## What This Project Covers

- Semantic HTML structure
- CSS Grid for layout
- Basic Flexbox usage
- Responsive layout principles
- Organizing files for a front-end project
